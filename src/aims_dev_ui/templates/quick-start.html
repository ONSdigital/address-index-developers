Quick Start

Quick start steps

  1. Request an Address Index account
  2. Explore the Address Index API

Request an Address Index account

To be able to explore and perform tests with the Address Index API, you need to request a username and password to use with each API call.

Explore the API

Each available Address Index API endpoint is listed on the Explore the API page. Each endpoint has its own reference page contains an embedded "explorer" that allows the reader to try out the service in-situ, without having to write any client code at all.
