Authorisation

Authentication

The Address Index API requires authentication credentials, currently the open test site has basic auth and other sites have JWT tokens and IP whitelists.

A username and password for basic auth will have been given to you by the AIMS support team.

Basic authentication requires the setting of the Authorization header to "Basic " + base64 encoded username and password.

One way to encode the username and password is via the F12 console in your browser: my_auth_string = "Basic " + window.btoa('YOUR_USERNAME:YOUR_PASSWORD')

Alternatively, many web frameworks allow the username and password to be passed as parameters when making the API call.

JWT requires the setting of the Authorization header to "Bearer " + token string.

Example

HTTP Basic Authentication

curl -XGET -u my_auth_string:
{{ host }}/addresses/uprn/10012133622

GET /addresses/uprn/10012133622 HTTP/1.1
Host: https://initial-test-open-api.aims.gcp.onsdigital.uk
Authorization: Basic bXldfdGGGG67BpX2eTo=
