Address Index API

Beta release

The ONS Address Index API (also known as AIMS - Address Index Matching Service) has been in beta for some time but is undergoing acceptance into service to become a fully live product.

Part of this service will include user support mechanisms to replace the old mailbox and slack channel.

A new version of the service (AIMS2) has just started development, but in the meantime improvements are still being made to AIMS1 including a new user UI and a bulk manager application.

Getting started

If you are new to the Address Index API, we recommend you take a look at the getting started guide, with the resources and examples provided there to help you.
