Sample Queries

Data limitations - the Exeter subset

This open version of the Address Index API Developers site runs against a subset of AddressBase, namely the Exeter free sample. As a result only addresses within the Exeter subset can be searched and returned by the 'Explore this endpoint' function.

To aid usage of the 'Explore the endpoint' functions, the below sample queries have been provided as a starting point.

Addresses

The following examples will return results from the /addresses endpoint when entered into the 'input' field

Gate Reach, Exeter

Chapel Street, Exeter

Market Street, Exeter

University Of Exeter

coffee

tesco

Partial

The following examples will return results from the /addresses/partial/{input} endpoint when entered into the 'input' field

7 Gat

4 EX26GA

Postcode

The following examples will return results from the /addresses/postcode/{postcode} endpoint when entered into the 'postcode' field

EX2 6GA

ex26ga

EX1 1LL
