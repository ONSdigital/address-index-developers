Authorisation

Authentication

The Address Index API requires authentication credentials, currently JWT tokens and IP whitelists.

To use the general-purpose API you need a whitelisted IP (ONS network already whitelisted) and a JWT token.

To use the bulk service from inside DAP, you need special API key for the CA Gateway.

JWT requires the setting of the Authorization header to "Bearer " + token string.
